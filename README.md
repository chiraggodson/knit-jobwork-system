# Knit Jobwork ERP System

Private ERP system for textile knitting factory operations.

## Modules
- Machine status dashboard
- Jobwork management
- Fabric tracking
- Production monitoring
- Yarn management

## Tech Stack

Backend
- Node.js
- Express
- PostgreSQL
- TypeORM

Frontend
- Flutter (Windows / Android / iOS)

## Project Structure

backend/ → API server  
flutter_app/ → ERP client application
